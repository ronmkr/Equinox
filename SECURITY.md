# Security Policy

## Supported Versions

Only the latest version of Equinox is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within Equinox, please do not open a public issue. Instead, please report it privately to the maintainers.

### Process
1.  **Email:** Send an email to [INSERT EMAIL ADDRESS] with details of the vulnerability.
2.  **Acknowledgement:** You will receive an acknowledgement within 48 hours.
3.  **Fix:** We will work on a fix and release a new version as soon as possible.
4.  **Disclosure:** Once the fix is released, we will publicly disclose the vulnerability (with credit to you, if desired).

Thank you for helping keep Equinox secure!
